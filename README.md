# Hotel Booking Website

## Overview
The Hotel Booking Website is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to browse hotel information, view room details, and complete a booking process with a success page.

## Features
- Responsive design
- Hotel details page
- Room booking interface
- Success confirmation page
- Interactive user interface
- Clean navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
Hotel_Booking/
│── index.html
│── Hotel.css
└── Hotel.js

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Or use the Live Server extension in VS Code
