# Hotel Booking Website

## Overview
The Hotel Booking Website is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to browse hotel information, view room details, and complete a booking process with a success page.

## Features
- Responsive design
- Hotel details page
- Room booking interface
- Success confirmation page
- Interactive user interface
- Clean navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript
